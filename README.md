# PBRPreview

## English

PBRPreview is a lightweight web-based application for **quick preview and validation of PBR (Physically Based Rendering) materials** without the need to open heavy 3D software.

This project is designed to help artists and developers perform a **quick PBR texture check** before integrating materials into the main production pipeline.

- Real-time PBR material preview  
- Supports standard PBR workflow:
  - Base Color / Albedo  
  - Roughness  
  - Metallic  
  - Normal  
- Clean and minimal interface focused on material inspection  
- Suitable for archviz, game assets, and general 3D workflows  

---

### üéØ Purpose

PBRPreview is created to:
- Validate PBR texture consistency and quality  
- Speed up material review workflows  
- Explore web-based material preview pipelines  
- Serve as a foundation for internal tools or open-source development  

---

### ‚öôÔ∏è How Does It Work?

PBRPreview is built as a **static web application** and runs entirely in the browser.

The application works by:
- Loading PBR texture files directly from local folders or user input  
- Rendering materials in real-time using browser-based rendering  
- Mapping each texture channel (Base Color, Roughness, Metallic, Normal) to a PBR material setup  
- Displaying the result instantly for visual inspection  

Since the project consists of **HTML, CSS, and JavaScript files only**, no backend or server setup is required.  
Simply open the HTML file in a modern browser to use the application.
