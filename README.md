# PBRPreview

PBRPreview is a lightweight web-based application for **quick preview and validation of PBR (Physically Based Rendering) materials** without the need to open heavy 3D software.

This project is designed to help artists and developers perform a **quick PBR texture check** before integrating materials into the main production pipeline.

---

PBRPreview adalah aplikasi ringan berbasis web untuk **preview dan validasi material PBR (Physically Based Rendering)** secara cepat tanpa perlu membuka software 3D yang berat.

Project ini dirancang untuk membantu artist dan developer melakukan **pengecekan cepat tekstur PBR** sebelum diintegrasikan ke dalam pipeline produksi utama.
